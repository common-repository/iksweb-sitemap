Плагин поможет быстро сгенерировать карту сайта sitemap.xml из существующих записей/страниц/рубрик/метов/страниц пользователей/пользовательских типов записей и даже таксономий.
 
== Description ==

XML Sitemap генератор - удобный плагин, который поможет без труда генерировать правильные карты сайта. Мы добавили самые необходимые настройки. Вы можете устанавливать частоту обновления, приоритеты, а самое главное - это всё можно делать как для целого раздела по умолчанию, так и для отдельных записей и страниц. Наш модуль также учитывает ваши пользовательские типы записей и позволяет отдельно их настраивать. 

Мы постоянно работает над усовершенствованием наших решений. Вы можете помочь нашему проекту и приобрести у нас платные версии плагинов.

Для маленьких сайтов (до 100 страниц) мы рекомендуем версию плагина 2.4 так как она более удобна в настройках. Версии 2.5 и выше предназначены для больших сайтов (500 и более страниц). 

= Особенности плагина =

На данный момент данный плагин является самым удобным и функциональным среди всех доступных решений. Лёгкий и компактный код не будет нагружать ваш сайт бесполезными функциями и скриптами. Только самое важное и нужное.

= О разработчике =

Плагины являются уникальной разработкой от создателей проекта IKSWEB.ru. Мы более 10 лет занимаемся создание и продвижением сайтов, а также разрабатываем плагины и различные решения.

== Frequently Asked Questions ==

= Зачем нужна карта сайта? =

Карта сайта необходима для составления списка страниц для поисковых систем. Она значительно упрощает индексирование сайта.

= Плагина бесплатный? =

Наш плагин совершенно бесплатный и весь функционал доступен всем пользователям из любых стран. Мы предоставляем Pro версию с улучшенной поддержкой и частыми обновлениями.

= Я могу прописывать свои правила в фильтрах к классу WP_Sitemap ? =

Конечно. Вы можете дополнительно устанавливать фильтры в файле function.php вашей темы.

== Installation ==

1. Установить и активировать плагин;
2. В настройках плагина включите активацию и укажите название файла карты сайта (По умолчанию sitemap);
3. Включите необходимые разделы и сохраните настройки;
4. После сохранения укажите ID записей и разделов, которые нужно исключить из карты;
5. После всех выставленных параметров, карта сайта будет сгенерирована.
6. Добавьте вашу новую кату в поисковые системы.
